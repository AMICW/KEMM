# KEMM 船舶专利摘要草案

本文档用于补齐正式申请所需的摘要材料。摘要以“技术方案要点 + 主要用途”为主，避免宣传性语言，提交前仍建议再做一次人工字数核对。

---

## 1. 发明名称

一种面向动态航行环境的船舶多目标轨迹优化方法、装置、电子设备及计算机程序产品

---

## 2. 推荐摘要版本

### 2.1 版本A：稳妥提交版

本发明涉及智能航运与船舶轨迹规划技术领域，公开了一种面向动态航行环境的船舶多目标轨迹优化方法。该方法获取本船、目标船、静态障碍物及环境场数据，构建以燃油消耗、航行时间和综合风险为目标并带有约束违背度的动态轨迹优化模型；根据历史前沿特征确定变化幅度、知识可迁移性和预测置信度，自适应分配记忆候选、预测候选、迁移候选和重初始化候选，构建统一候选池并执行约束支配环境选择，确定代表轨迹用于滚动重规划。该方法用于提升动态航行场景中的规划安全性、稳定性和恢复效率。

推荐原因：

- 技术问题、技术方案和用途都已经交代。
- 语言相对稳，不容易因写得过学术而偏离摘要口径。
- 与 [说明书简版文本](d:/VS%20Code/code/KEMM%28renew%29/docs/patent_specification_ship_kemm_draft.md#L1517) 一致性高。

### 2.2 版本B：偏方法流程版

本发明公开了一种面向动态航行环境的船舶多目标轨迹优化方法。该方法获取本船、目标船、静态障碍物及环境场数据，建立以燃油消耗、航行时间和综合风险为目标的动态轨迹优化模型；根据当前规划周期与历史规划周期的前沿特征确定环境变化信息，并据此分配历史记忆候选、前沿预测候选、跨环境迁移候选和重初始化候选的生成比例；将各类候选解构成统一候选池，执行约束支配环境选择，得到当前轨迹解集并选取代表轨迹用于滚动重规划。该方法可用于船舶动态避碰与路径规划。

适用情形：

- 更强调步骤链。
- 适合方法权利要求主线已经固定时使用。

### 2.3 版本C：偏应用场景版

本发明涉及船舶自动避碰和路径规划技术领域，公开了一种面向动态航行环境的船舶多目标轨迹优化方法。针对目标船状态、静态障碍物和环境场随时间变化导致的轨迹恢复慢和安全性下降问题，本方法通过环境变化诊断、自适应候选分配、多源候选生成、统一候选池构造以及约束支配环境选择，获得满足安全优先原则的代表轨迹，并在滚动重规划框架下输出控制结果。该方法适用于近海会遇、港口受限水域和高密障碍环境中的船舶航行规划。

适用情形：

- 更突出应用价值。
- 适合老师或代理人希望摘要里更明确写出船舶场景时使用。

## 5. 摘要撰写注意事项

1. 摘要文字部分控制在 `300` 字以内，最终以人工逐字统计为准。
2. 摘要中不建议出现：

- “显著提高”
- “大幅提升”
- “明显优于”
- “首次提出”

这些更适合写进技术效果，不适合正式摘要。

3. 摘要中一般不建议写过多具体公式和超参数。
4. 摘要应以技术方案为主，效果和用途为辅。
5. 摘要最好不要直接写：

- `VAE`
- `GP`
- `Grassmann`

除非代理人明确希望用这些术语帮助区分现有技术。
